    Authentication functionality

    OSJO Authentication is handled by JWT. First make a request to the login route with a username and password. THis will return you a JWT that is valid for the next 60 minutes. Include this as the Authorization header in your requests e.g "Authorization: Bearer authTokenhere"

    Get a JWT via given credentials.

    Example request:

    curl -X POST "http://localhost/api/auth/login" 
    const url = new URL("http://localhost/api/auth/login");

let headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
}

    method: "POST",
    headers: headers,
